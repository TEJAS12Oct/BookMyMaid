DATABASE TABLES INFORMATION
NAME , PRIMARY & FORIEGN KEY 


1. ADMIN INFO TABLE
   ID AUTO_INCREMENT (PRIMARY KEY) , USERNAME , PASSWORD
  
2. MAID INFO TABLE
   ID AUTO_INCREMENT (PRIMARY KEY) , NAME , USERNAME ,PASSWORD , GENDER , DOB , 
   MOBILE_NO , EMAIL_ID , ADDRESS , ADHAR_CARD , POLICE_VERIFICATION_CERTIFICATE , 
   EXPERIENCE , RATING , IMAGES  

3. USER INFO TABLE
   ID AUTO_INCREMENT (PRIMARY KEY) , NAME , USERNAME , PASSWORD, 
   GENDER , MOBILE_NO , EMAIL_ID, ADDRESS , IMAGES

4. SERVICES TABLE
   ID AUTO_INCREMENT (PRIMARY KEY) , SERVICE , CHARGES

5. BOOKING INFO
   ID AUTO_INCREMENT (PRIMARY KEY) , SERVICE , CHARGES , 
   MAID_ID REFERENCE (MAID INFO TABLE) ID (FORIEGN KEY), 
   MAID_NAME REFERENCE (MAID INFO TABLE) NAME (FORIEGN KEY) , 
   USER_ID REFERENCE (USER INFO TABLE) ID (FORIEGN KEY), 
   USER_NAME REFERENCE (USER INFO TABLE) NAME (FORIEGN KEY) ,
   TIME (START & END) , DATE (START & END)

6. MAID REVIEW (FEEDBACK)
   ID AUTO_INCREMENT (PRIMARY KEY) , 
   MAID_ID REFERENCE (MAID INFO TABLE) ID (FORIEGN KEY),  
   USERNAME REFERENCE (USER INFO TABLE) USERNAME (FORIEGN KEY) ,
   RATING  , DESCRIPTION

7. CONTACT US
   ID AUTO_INCREMENT (PRIMARY KEY) , NAME , MOBILE_NO , EMAIL_ID , CITY , MESSAGE
